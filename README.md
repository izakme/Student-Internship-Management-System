Student Internship Management System

Project OverView

The student internship Management System is a web-based application developed 
using PHP Object-Oriented Programming and MySQL. The system automates
internships management processes by connecting students, companies, and 
administrators on a centralized platform.

Features

Student:
  Register and Login
  View Internships
  Search Interships
  Apply for Internships
  Track Appliction status

Company:
  Register and Login
  Post Internship Opportunities
  Manage Internship Listings
  Review Apllications

Administrator
  Manage Users
  Manage Applications
  Generate Reports
  Monitor System Activities

Technologies that will be Used
  PHP(OOP)
  MySQL
  HTML
  CSS
  JavaScript
  PDO
  Apache
  AWS EC2

Security Features
  Password Hashing
  Session Management
  Prepared Statements

Database design
  3NF- Aiming to reduce data redundancy and improve data integrity.
